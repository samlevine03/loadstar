# Loadstar

Loadstar is a social media app with first-class lift tracking for people who already know what they are doing in the gym.
MVP0 is the offline-first tracking stack inside a socially native 5-tab app shell: `Home`, `Explore`, `Lift`, `History`, and `Profile`.
The `Lift` tab is the center primary action and should behave like an active recording surface: if a workout is in progress, reopening the app should return the user directly to it.
Loadstar is not a trainer, coaching product, or prescriptive workout app. It records what the user actually did and makes that history social, durable, and easy to revisit.
The docs also cover the broader product direction toward the lifting equivalent of Strava.
